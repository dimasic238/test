Тренировка работы с ветками

Нажимайте кнопку, меняйте фон и практикуйте Git!

Количество нажатий:

0

Инструкция:

  1. Нажмите кнопку выше несколько раз
  2. Измените код (например, добавьте новую кнопку)
  3. Создайте новую ветку в Git
  4. Запушьте изменения на GitHub
  5. Создайте Pull Request
